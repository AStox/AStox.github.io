Knot Fun

500,000+ Downloads!

Knot Fun is a pseudo-puzzle mobile game about untying knots. Really. It randomly generates a knot of varying difficulty and lets you tug and tease it out in a variety of styles including headphones, christmas lights, garden hose, and more! Developed with Unity and Gimp and written entirely in C#. It's available to download for free on both Android and iOS app stores.

Hey! Listen (v2.0)

Hey! Listen is now primarily a collaborative internet radio app. Building on the the original idea of collaborative playlists, users can join a channel and listen to their playlist syced up in real-time with other connected users as well as chat about it. It uses Youtube's API to query videos, and the video.js library to embed Youtube iframes. Built on Ruby on Rails, Coffeescript, Javascript, and Foundation. The video syncing functionality and chat room use websockets.

Hey! Listen (v1.0)

Hey! Listen is a collaborative playlist web app. I built it as a way for my friends and I to share new music. Hey! Listen uses Youtube's API to search for music, and adds songs to a dynamic playlist. It's still a work in progress but it is currently functional and has an interesting assortment of music on it. Built with Ruby/Rails, HTML5, CSS3, Javascript, Jquery, and lots of AJAX.
